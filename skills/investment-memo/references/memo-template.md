# Investment Memo Template — Quick Reference

## Section Flow
COVER → CONVICTION → COMPANY → TEAM → MARKET → TERMS → RISKS → RECOMMENDATION

## Pull Quote Examples
- Conviction: "Software ate the world. AI ate software. Now [X] will eat [Y]. And every [X] needs [Z] first."
- Team: "In venture, you bet on the jockey. These [N] have already won [M] races."
- Market: "[Industry leader quote with attribution]"
- Recommendation: "[Metaphor that captures the thesis in one line]"

## Conviction Pattern
1. Name the paradigm shift (one sentence)
2. Name the bottleneck (one sentence)
3. Name the company as the solution (one sentence)
4. Connect to Triptyq thesis (one paragraph, end bold red italics)

## Founder Entry Format
`[Name in RED bold] — [Best exit]. [Second credential]. [Data room evidence]. [Role].`
Lead with exits, not titles. Cite documents as proof.
